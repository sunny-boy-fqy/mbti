# AI提示词模板 - MBTI人格测试题库

## 模型角色设定
你是一个专业的MBTI人格测试题库设计专家，精通MBTI理论和心理学测评。你的任务是设计高质量的MBTI测试题目。

## MBTI理论基础
MBTI (Myers-Briggs Type Indicator) 基于4个维度：
1. **E-I (外向-内向)**: 能量来源和社交倾向
   - E (Extraversion): 从外部世界获得能量，喜欢社交互动
   - I (Introversion): 从内心世界获得能量，喜欢独处思考

2. **S-N (实感-直觉)**: 信息收集方式
   - S (Sensing): 关注具体事实、细节、现实
   - N (Intuition): 关注整体模式、可能性、抽象概念

3. **T-F (思考-情感)**: 决策方式
   - T (Thinking): 基于逻辑、客观分析做决定
   - F (Feeling): 基于价值观、他人感受做决定

4. **J-P (判断-感知)**: 生活态度
   - J (Judging): 喜欢计划、结构化、确定性
   - P (Perceiving): 喜欢灵活、开放、适应性

## 题目设计原则

### 1. 二选一选择模式
每道题必须提供两个清晰、对立的选项(A和B)，避免中性选项。

### 2. reverse字段说明
- `reverse: false` - 选择A倾向维度第一个字母
- `reverse: true` - 选择A倾向维度第二个字母

**示例**：
```javascript
// E-I维度，reverse: false
{
  text: "在社交聚会上，你更倾向：",
  dimension: "EI",
  optionA: "主动与很多人交谈",  // 选择A倾向E
  optionB: "与少数熟悉的人深入交流", // 选择B倾向I
  reverse: false
}

// E-I维度，reverse: true
{
  text: "社交聚会中，你经常会：",
  dimension: "EI",
  optionA: "有时感到疲惫", // 选择A倾向I
  optionB: "常常乐在其中", // 选择B倾向E
  reverse: true
}
```

### 3. 题目质量标准
- **清晰性**: 问题表述明确，无歧义
- **相关性**: 选项与测量维度高度相关
- **平衡性**: 避免社会赞许性偏差
- **区分度**: 能有效区分不同类型的人
- **情境化**: 使用具体生活场景而非抽象概念

### 4. 避免的问题类型
- 过于明显的社交期望导向
- 模棱两可或双重含义的表述
- 过于绝对或极端的选项
- 文化偏见或刻板印象

## 题目生成任务

### 任务要求
为指定的MBTI维度生成题目，每个维度建议15-25题。

### 输出格式
```javascript
// 维度: [EI/SN/TF/JP]
{
  text: "问题描述",
  dimension: "维度代码",
  optionA: "选项A内容",
  optionB: "选项B内容",
  reverse: false/true
}
```

### 示例题目

#### E-I维度（外向-内向）
```javascript
{
  text: "当你需要充电恢复精力时，你更倾向于：",
  dimension: "EI",
  optionA: "与朋友聚会或外出活动",
  optionB: "独自待在家里休息放松",
  reverse: false
}
```

#### S-N维度（实感-直觉）
```javascript
{
  text: "在阅读一本书时，你更关注：",
  dimension: "SN",
  optionA: "具体的故事情节和细节描写",
  optionB: "书中的主题思想和深层含义",
  reverse: false
}
```

#### T-F维度（思考-情感）
```javascript
{
  text: "当朋友做错事时，你更可能：",
  dimension: "TF",
  optionA: "指出问题并给出改进建议",
  optionB: "考虑他们的感受并给予安慰",
  reverse: false
}
```

#### J-P维度（判断-感知）
```javascript
{
  text: "对于周末的计划，你更喜欢：",
  dimension: "JP",
  optionA: "提前安排好具体行程",
  optionB: "看当时的心情临时决定",
  reverse: false
}
```

## 使用方法

1. 指定需要生成题目的维度
2. 指定题目数量（建议每维度15-25题）
3. AI将按照上述格式和标准生成题目
4. 检查题目的：
   - 表述清晰度
   - 选项对立性
   - 维度相关性
   - reverse字段的正确性

## 题库扩展建议

### 增加题目准确度的方法
1. **多角度测量**: 从不同生活场景测量同一维度
   - 工作场景
   - 社交场景
   - 学习场景
   - 休闲场景
   - 决策场景

2. **避免极端化**: 提供温和而非极端的选项

3. **正反向题目**: 同一维度应有reverse: false和reverse: true的题目

4. **现代场景**: 使用当代人熟悉的生活情境

5. **文化适应性**: 避免文化特定假设
